# Organize Photos
* Very simple effective program to organize photos in a folder by date order
* It has simple gui support
* Supported filetypes:
* ['.jpg', '.JPG', '.JPEG', '.jpeg', '.png', '.PNG', '.JFIF', '.jfif', '.TIFF', '.tiff', '.tif', '.TIF']
